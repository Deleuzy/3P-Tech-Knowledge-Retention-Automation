# Equipment Failure Analysis

## Copy/Paste This Prompt

---

I need to analyze an equipment failure:

**EQUIPMENT:**
- Type: [electrical/mechanical/pneumatic/hydraulic/other]
- Model/Specs: [as much as you know]
- Location: [where is it installed]
- Age: [how old/how long in service]

**FAILURE DESCRIPTION:**
- What happened: [describe the failure mode]
- When it happened: [timing - sudden/gradual/under load/at rest]
- Observable symptoms: [what you see/hear/measure]
- Impact: [does it affect operations/safety/both]

**CONTEXT:**
- Normal operating conditions: [describe]
- Conditions when it failed: [what was different]
- History: [first time/recurring/pattern]
- Related failures: [any similar equipment having issues]

**QUESTION:**
1. What caused this failure?
2. How serious is it?
3. Can it be repaired or must be replaced?
4. How do we prevent it happening again?
5. How urgent is this?

Please provide:
- Root cause analysis
- Failure severity assessment
- Recommended action (repair/replace/monitor)
- Prevention measures
- Urgency level (immediate/soon/scheduled)

---

## Notes

- Describe observable facts, not guesses
- Include any unusual conditions present
- Note if equipment is under warranty
- Mention if similar equipment is failing
