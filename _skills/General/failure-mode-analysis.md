# Failure Mode Analysis

## Copy/Paste This Prompt

---

I need to analyze failure modes for a system or equipment:

**SYSTEM/EQUIPMENT:** [What are we analyzing?]

**SYSTEM COMPONENTS:**
- [Component 1]
- [Component 2]
- [Component 3]
- [etc.]

**FAILURE MODES TO CONSIDER:**
- [Failure 1: e.g., "electrical short"]
- [Failure 2: e.g., "mechanical breakage"]
- [Failure 3: e.g., "corrosion"]
- [Failure 4: e.g., "seal failure"]
- [etc.]

**CONTEXT:**
- Operating environment: [Temperature/Humidity/Vibration/etc.]
- Usage intensity: [Heavy/Normal/Light]
- Age: [How long in service]
- Maintenance: [How often/What's done]
- History: [Any previous failures?]

**REQUIREMENTS:**
- Uptime needed: [%/hours per day]
- Safety critical: [Yes/No/Partial]
- Cost of failure: [Economic/Safety/Both]
- Repair time available: [Immediate/Days/Weeks]

**QUESTIONS:**
1. What are the most likely failures?
2. What would cause each failure?
3. How would each failure manifest?
4. What's the impact of each failure?
5. How can each be prevented?
6. How can each be detected early?
7. What's the priority to prevent?

Please analyze each failure mode:
- Root causes (why it fails)
- Probability of occurrence
- Consequence severity
- Current detection capability
- Current prevention methods
- How early can it be detected?
- Impact on operations/safety

Provide:
- List of all failure modes
- Severity ranking (critical/major/minor)
- Probability ranking (high/medium/low)
- Root causes for each
- Prevention recommendations
- Detection/monitoring recommendations
- Maintenance recommendations
- Priority action items

---

## Notes

- Be thorough about possible failure modes
- Consider environmental factors
- Include maintenance history
- Note past failures if any
- Consider redundancy/backup systems
- Think about operator errors too
